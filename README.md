# Telegram Digital Code Bot

A Telegram bot for selling digital codes with:
- User account system
- Purchase history with dates
- SQLite database
- Admin approval system
- Auto code delivery
- Copy card number button

## Run locally:
pip install -r requirements.txt
python bot.py

## Deploy on Render:
- Create new Web Service
- Build command: pip install -r requirements.txt
- Start command: python bot.py
